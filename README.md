# Проект bank_prj
## Описание
Проект bank_prj - это виджет, который показывает несколько последних успешных банковских операций клиента.
## Установка
Клонируйте репозиторий:
```
git clone git@github.com:VeronikaSidorova/bank_prj.git
```
## Использование
1. Откройте приложение в вашем веб-браузере.
2. Введите реквизиты счета или карты для получения последних успешных операций.
3. Просмотрите результат
4. Приложение умеет фильтровать транзакции по указанной валюте или выводить описание транзакции
5. Есть возможность сгенерировать номера карт в заданном диапазоне
6. Разработан декоратор log, который автоматически регистрирует детали выполнения функций, такие как время вызова, имя функции, передаваемые аргументы, результат выполнения и информация об ошибках
7. Программа умеет считывать транзакции из файлов типа csv и excel.
## Тестирование
Наш проект покрыт тестами. Для их запуска выполните команду:
```
pytest
```
